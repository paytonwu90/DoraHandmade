# 2025 React 專題班-愛哆啦也愛手作 | DoraHandmade

**線上預覽：** https://paytonwu90.github.io/DoraHandmade/

## 創作理念與初衷

源於朋友在空閒時間手作蝴蝶結的熱情。 過去依賴大型電商平台與FB 粉專 卻面臨功能限制多 、 手續費高昂且缺乏直接接觸顧客管道的困境 。
「建立結合募資與電商功能的品牌網站，展示手作溫度的同時，解決創作者的經營痛點 。」
我們致力於打造一個專屬於手作藝術家的空間，讓精緻的緞帶原料、季節限定系列與客製化服務，能以最直覺、優雅的方式呈現給每一位喜愛手作的顧客。

## 現有平台痛點與解決方案

- 低成本經營：擺脫手續費枷鎖
  解決蝦皮等平台不斷調漲的手續費與強制轉嫁的運費負擔建立透明且可控的成本結構。
- 品牌聚焦：建立直連溝通管道
  解決 FB 社團功能零散與資訊混雜問題，透過整合式後台與顧客建立深度連結，累積品牌會員。
- 體驗優化：極簡直覺的UX

## 開發技術棧與視覺風格

- 前端與版本控制：React.js、Bootstrap、Git
- 後端與安全性：PHP、MySql、JWT (JSON Web Token)
